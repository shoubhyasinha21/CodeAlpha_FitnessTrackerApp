# CodeAlpha_FitnessTrackerApp
Fitness Tracker App
Overview

The Fitness Tracker App is a simple and user-friendly web application that allows users to track their daily fitness activities. Users can log exercises, workout duration, and calories burned, and visualize their progress in an easy-to-read dashboard. This project demonstrates connecting a frontend interface to a backend (Firebase) for storing and retrieving user data in real-time.

Features

Log daily fitness activities manually (exercise type, duration, calories burned).

Dashboard summarizing daily workouts, total minutes, and total calories burned.

Activity history displayed in a table format.

Real-time updates of dashboard when new activities are added.

Clean, responsive UI with cards and statistics.

Backend integration with Firebase Firestore for storing activity records.


Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Firebase Firestore (for storing activities)

Optional: Local storage (can replace Firebase for offline usage)
